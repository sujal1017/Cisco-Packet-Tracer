# Cisco Packet Tracer Practice Labs

This repository contains my Cisco Packet Tracer (`.pkt`) practice files created while learning Computer Networks and Routing concepts.

The purpose of this repository is to document hands-on practice with network design, configuration, and troubleshooting using real CLI-based commands.

---

## 📚 Topics Covered

* Basic LAN Setup
* IP Addressing & Subnetting
* Router Configuration
* Switch Configuration
* Inter-LAN Communication
* Static Routing
* Default Gateway Configuration
* Interface Configuration (FastEthernet, Serial)
* Packet Simulation & PDU Testing

---

## 🖥 Tools Used

* Cisco Packet Tracer (1841 Routers, 2960 Switches)
* CLI-based Configuration
* Simulation Mode for Packet Analysis

---

## 🗂 Repository Structure

Each `.pkt` file represents a different lab experiment or practice scenario.

### Example Labs:

* Single Router with Multiple LANs
* Two Router Static Routing Setup
* Three Router Network Topology
* RIP Configuration Between Routers
* Basic Network Topology Setup
* LAN to LAN Communication

---

## 🎯 Learning Objectives

* Strengthen understanding of routing concepts
* Practice real-world CLI commands
* Improve troubleshooting skills
* Prepare for networking exams and practical assessments

---

## 🚀 How to Use

1. Download any `.pkt` file
2. Open it using Cisco Packet Tracer (version 8.x or above recommended)
3. Explore the topology and configurations
4. Run simulations to understand packet flow
